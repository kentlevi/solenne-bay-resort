# Solenne Bay Luxury Resort

A premium, editorial-grade landing page for Solenne Bay, a high-end luxury resort. This application is built with modern web technologies to provide a cinematic and highly responsive user experience.

## ✨ Features

- **Cinematic Entry Animations**: Powered by **GSAP** for a sophisticated first impression.
- **Editorial Layouts**: Custom-designed sections tailored for a luxury brand aesthetic.
- **Adaptive Components**: Smooth sliders and room showcases using **Swiper**.
- **Modern Tech Stack**: Built with **Nuxt 3** and **Tailwind CSS** for maximum performance and flexibility.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Framework**: [Nuxt 3](https://nuxt.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://gsap.com/)
- **Icons**: [Lucide Vue Next](https://lucide.dev/guide/packages/lucide-vue-next)
- **Slider**: [Swiper](https://swiperjs.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm, pnpm, yarn, or bun

### Setup

Install dependencies:

```bash
# npm
npm install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

### Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
